Paint VR - Vive la Realidad Virtual pintando

Paint VR es un juego de realidad virtual que permite a los usuarios experimentar el arte de la pintura en un entorno VR. A través de este juego, los jugadores pueden intentar recrear con precisión obras maestras o cuadros originales. El objetivo es lograr una réplica lo más real posible y superar cada reto artístico, brindando una experiencia inmersiva y satisfactoria para los amantes de la pintura.

Proyecto

Relax Garden - Una Ayuda para Organizarse en la Universidad

Relax Garden es una herramienta interactiva diseñada para apoyar a estudiantes universitarios en su organización. A través de un pequeño huerto virtual, los estudiantes pueden aprender a gestionar su tiempo y reducir el estrés, cuidando de sus plantas en sesiones breves y gratificantes. Con un ambiente visual y sonoro relajante, este juego ayuda a los nuevos estudiantes a adquirir hábitos de organización y bienestar mientras se adaptan a la vida universitaria.

Explora Relax Garden
  • CREACION DE LA IDEA
    Inicialmente teniamos la idea de crear un piano en VR, donde el jugador podria tocar libremente o tambien en base a alguna cancion de su eleccion. Planteamos tareas que el usuario podria realizar, condiciones de victoria y de derrota.
  • SKETCHING
    Se bocetaron algunas vistas que el usuario iba a tener en el juego, asi como las mecanicas que este iba a tener y posibles interacciones con el usuario.
  • PROTOTIPO / MAQUETA Y EVALUACION
    Se creo una maqueta interactuable, la cual los usuarios pudieron probar y de ese modo pudimos recopilar feedback de como ellos interactuaban con el videojuego, si entendian el objetivo o aun no estaba claro, etc.
  • EVOLUCION DE LA IDEA
    Dimos otro enfoque a la idea de nuestro videojuego, ahora el usuario es un pintor, tendra diferentes dificultades y dependiendo de la habilidad del usario ganara mas o menos puntos.
  • PRIMER PROTOTIPO
    Se hizo una evaluacion con diferentes usuarios, de modo que, pudimos observar como ellos interactuan con el ambiente, algunas dificultades que tienen y que tareas se podrian modificar .
  • SEGUNDO PROTOTIPO
    En base a la retroalimentacion de los usarios, se realiza diferentes moficaciones y se pudo observar una mejora, de igual manera se continuo la evaluacion con los usuarios para poder observar que otras cosas se podrian mejorar.
  • TERCER PROTOTIPO
    Se implementaron principios de gestalt a la interfaz del usuario.
Herramientas Usadas

Explora las Herramientas Utilizadas en el Proyecto

Unity

Motor de desarrollo que permite la creación de experiencias VR interactivas, utilizado para construir y animar el entorno 3D.

C#

Lenguaje de programación utilizado para desarrollar la lógica del juego, con scripts que controlan las interacciones y la mecánica de juego.

Blender

Software de modelado 3D usado para crear y texturizar los objetos y elementos visuales, aportando realismo a los escenarios de VR.

HTML

Lenguaje de marcado usado para estructurar la interfaz web, proporcionando una base sólida para la visualización del proyecto en línea.

Bootstrap

Framework CSS que facilita el diseño responsivo, permitiendo que la interfaz web se adapte adecuadamente a diferentes dispositivos.

jQuery

Biblioteca de JavaScript utilizada para simplificar interacciones y efectos en la página web, mejorando la experiencia de usuario.
